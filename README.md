# Bitasmbl-Developer-Portfolio-Landing-Page-356a19-3cfc00c7448f

## Description
A responsive single-page portfolio site for a junior web developer, featuring hero, about, projects, and contact sections, built with HTML and Tailwind CSS.

## Tech Stack
- Java

## Requirements
- Create a responsive hero section with developer name, title, and primary call-to-action button.
- Implement an About Me section with a short paragraph and an image placeholder that stacks on mobile and aligns side-by-side on larger screens.
- Display three featured projects with titles, descriptions, and GitHub links in a responsive grid.
- Build a contact section with a simple form including name, email, and message fields.
- Add simple hover animations for interactive elements like buttons or project cards.

## Installation
- git clone https://github.com/he1snber8/Bitasmbl-Developer-Portfolio-Landing-Page-356a19-3cfc00c7448f.git
- cd Bitasmbl-Developer-Portfolio-Landing-Page-356a19-3cfc00c7448f

## Usage
- Open the project in your Java-capable environment and serve static files as configured.

## Implementation Steps
1. Set up a Java project structure to host static HTML assets.
2. Create the main HTML page with sections for hero, about, projects, and contact.
3. Integrate Tailwind CSS classes into the HTML for responsive layout.
4. Implement the hero section with developer name, title, and call-to-action button.
5. Build the About Me section with text and image placeholder responsive behavior.
6. Add three project entries with titles, descriptions, and GitHub links in a grid.
7. Implement the contact form with name, email, and message fields.
8. Add CSS-based hover states using Tailwind utility classes for buttons and project cards.
